1) Разархивировать папку на свой компьютер
2) Открыть SQL Server Management Studio
3) Экспортировать из папки в Sql файл .backpack с помощью: импорта приложения уровня данных
4) Зайди в папку :CompHomp-CompHomp- и открыть CompHomp.sln
5)После загрузки всех файлов найти файлы :AppDbContext.cs,App.xaml.cs и MainUserWindow.xaml.cs
6) Найти строку подключения:"Data Source=HOME-PC;Initial Catalog=CompHompDb;Integrated Security=True;Connect Timeout=30;Encrypt=False;Trust Server Certificate=False;Application Intent=ReadWrite;Multi Subnet Failover=False"

Взять строку подключения из SQL Server Management Studio и вставить в Data Source=……
7)Можно успешно возвращаться и запускать через wpf по кнопки:Запуск CompHomp
8)Исходные данные: 
Вход для Администратора:admin(логин) admin123(пароль) и можете пользоваться возможностями админа
Вход для пользователя через регистрацию.
